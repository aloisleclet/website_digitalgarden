# website_digitalgarden

minimal bash + html digital garden website.

## Install

```bash
git clone https://github.com/aloisleclet/website_digitalgarden
cd website_digitalgarden
chmod +x ./install
./install
```

## How to use

1. Post

```bash
garden post
```

2. Reset

drop all posts

```bash
garden reset 
```

3. Show all posts

```
garden history
```

code with <3
